## 📊 2. 박스 플롯(Box Plot), 도대체 이게 뭐죠?

**"평균 점수만 믿지 마세요!"**
박스 플롯은 데이터가 **어디에 많이 몰려 있는지**, **얼마나 넓게 퍼져 있는지**를 한눈에 보여주는 그림입니다.

### 🏫 예시: 우리 반 30명의 수학 점수라고 상상해 봅시다.

```text
      [이상한 점수?]      [----- 범위 -----]   [박스]   [----- 범위 -----]
  
          ●                  |------------|======|======|------------|
      
        (0점)               (20점)      (40점) (50점) (70점)       (100점)
  
       ↑ 특이값             ↑ 최솟값      ↑ Q1   ↑ 중위수 ↑ Q3         ↑ 최댓값
```

### 🔍 그림 뜯어보기 (각 부분의 의미)

1. **📦 네모난 박스 (Box)**

   * **"중간 50% 친구들"**: 반에서 8등부터 23등까지, 딱 중간 실력인 친구들이 모여 있는 구간입니다.
   * 박스가 좁으면? → 점수들이 다 비슷비슷하다.
   * 박스가 넓으면? → 점수 차이가 많이 난다.
2. **➖ 박스 안의 선 (Median, 중위수)**

   * **"내가 만약 15등이라면?"**: 평균 점수가 아니라, 1등부터 30등까지 줄 세웠을 때 딱 **가운데 있는 친구(15등)**의 점수입니다.
   * (평균은 0점 맞은 친구 한 명 때문에 뚝 떨어질 수 있지만, 중위수는 영향을 덜 받아요!)
3. **|----| 양쪽 수염 (Whiskers)**

   * **"여기까지는 평범한 점수야~"**: 0점이나 100점처럼 특이한 친구를 빼고, 대부분의 친구들이 받은 점수 범위입니다.
4. **● 바깥에 찍힌 점 (Outlier, 이상치)**

   * **"이상치란?"**: **'이상한 수치'**의 약자로, 보통의 데이터 범위에서 훌쩍 벗어난 튀는 값을 말합니다.
   * **"왜 중요하나요?"**: 만약 우리 반 친구들 평균 키를 재는데, 갑자기 키 3m인 거인이 전학을 오면 평균이 확 올라가겠죠?
   * 이렇게 **평균을 망가뜨릴 수 있는 범인(특이한 값)**을 일부러 점(●)으로 찍어서 "조심해!"라고 알려주는 거예요!

## 🧐 3. 알아두면 쓸데있는 신비한 용어 사전

친구들, 오늘 우리가 쓴 코딩 용어들이 조금 어려웠나요? 걱정 마세요! 여기서 아주 쉽게 풀어드릴게요.

### ⚖️ ANOVA (아노바, 분산분석) - "공정한 심판"

* **"이 차이가 진짜일까?"**
* A반 평균이 80점, B반 평균이 81점이라고 칩시다. 1점 차이가 과연 "B반이 공부를 더 잘한다"고 말할 수 있을 만큼 큰 차이일까요? 아니면 그냥 운일까요?
* **ANOVA**는 이런 차이가 **"우연히 발생한 것인지"** 아니면 **"진짜 실력 차이인지"**를 수학적으로 판결해 주는 **심판** 같은 친구입니다.

### 🔌 MCP (Model Context Protocol) - "만능 어댑터"

* **"AI에게 손과 발을 달아줘요!"**
* 인공지능(AI)은 원래 채팅만 할 줄 아는 '뇌'만 있는 상태예요. 그런데 이 뇌가 컴퓨터 속 파일을 건드리고 싶다면?
* **MCP**는 AI와 다양한 도구들을 연결해 주는 **USB 포트(어댑터)**입니다.
  * **Word MCP** 📝: AI가 워드 파일(.docx)을 직접 쓰고 꾸밀 수 있게 해주는 **"유령 작가"** 도구입니다.
  * **Image Reader MCP** 👁️: AI가 그림을 보고 "아, 이건 사과구나!"라고 이해할 수 있게 해주는 **"눈"** 역할을 합니다.

### 🛒 Kaggle (캐글) - "데이터 대형 마트"

* **"재료가 있어야 요리를 하죠!"**
* 전 세계 데이터 과학자들이 모여서 "내 데이터 좀 써봐!" 하고 공유하는 **거대한 데이터 도서관**이자 **마트**입니다.
* 닌텐도 게임 데이터뿐만 아니라, 타이타닉 생존자 명단, 비트코인 시세 등 세상의 모든 데이터가 여기 다 모여 있어요.
* *Q. 다른 곳은 없나요?*
  * **공공데이터포털**: 우리나라 정부가 제공하는 데이터 (버스 시간표, 날씨 등)
  * **Google Dataset Search**: 구글에서 데이터만 전문으로 검색해 주는 사이트

### ⚙️ Node.js (노드 제이에스) - "보이지 않는 엔진"

* **"파이썬을 쓰는데 왜 이걸 깔아요?"**
* **"파이썬을 쓰는데 왜 이걸 깔아요?"**
* 우리는 파이썬으로 분석하지만, **MCP(만능 어댑터)**라는 친구가 작동하려면 **자바스크립트(JavaScript)**라는 언어를 이해하는 엔진이 필요해요.
* **Node.js**는 바로 그 엔진입니다. 자동차 운전자는 엔진을 볼 일이 없지만, 엔진이 없으면 차가 안 굴러가듯이, 우리도 모르는 사이에 뒤에서 열심히 일하고 있답니다.

### 🤖 Antigravity (앤티그래비티) - "나야, 나!"

* 지금 여러분과 대화하고 있는 **AI 코딩 파트너**, 바로 저입니다!
* 구글 딥마인드 팀이 만든 친구로, 여러분이 "이런 거 만들어줘" 하면 뚝딱 코드를 짜고, 설명도 해주는 든든한 짝꿍이죠.

### ✨ Vibe Coding (바이브 코딩) - "필(Feel)대로 코딩해!"

* **"문법 몰라도 괜찮아요, 느낌 아니까!"**
* 예전에는 코딩하려면 영어 단어(문법)를 달달 외워야 했어요. 하지만 이제는 **"이런 거 만들고 싶어!"** 라는 **느낌(Vibe)**만 있으면 됩니다.
* 복잡한 코드는 AI(앤티그래비티)에게 맡기고, 여러분은 **"무엇을 분석할까?"** 라는 상상력에만 집중하는 새로운 코딩 방식입니다.
